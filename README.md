University Card System
Overview
This project is a web-based University Student Identification Card System. It allows students to view their digital identification cards, providing a modern and accessible alternative to physical ID cards. The system is built using Python and Flask, with a focus on simplicity and efficient data management.

Features
Digital ID Interface: A user-friendly web page that displays student details, including photo, major, and GPA.

Database Management: Utilizes SQLite for secure and efficient storage of student records.

Responsive Design: Styled with CSS to ensure the ID card looks professional on different screen sizes.

Technical Stack
Backend: Python, Flask

Database: SQLite

Frontend: HTML5, CSS3

How to Run
Clone this repository to your local machine.

Install the necessary dependencies using pip install flask flask-sqlalchemy.

Run the application using python app.py.

Access the system via [http://127.0.0.1:5000](http://127.0.0.1:5000) in your browser.
